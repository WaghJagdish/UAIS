Count-Min Sketch Guarantees

Count-Min Sketch is a probabilistic data structure used to estimate frequencies. It uses a fixed size 2D matrix of counters of depth d and width w. It guarantees that estimated count does not exceed true count by more than ε · N with probability at least 1 - δ, where:

  • ε (error margin) = e / w
  • δ (error probability) = e-d

Reservoir Sampling Theorem

Reservoir sampling is an algorithm for choosing a simple random sample of size k from a stream of unknown or infinite length. For the i-th item in the stream:

  • If i < k, add directly to reservoir.
  • If i ≥ k, generate a random index j in [0, i]. If j < k, replace reservoir[j] with the item.
  • At the end of stream, each item has an equal probability k / N of being in the final sample.
