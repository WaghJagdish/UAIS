Dijkstra's Algorithm

Dijkstra's algorithm finds the shortest path from a single source node to all other nodes in a weighted graph with non-negative edge weights. It uses a priority queue (min-heap) to repeatedly extract the node with the minimum tentative distance, visits its neighbors, and relaxes their edge distances.

Kruskal's MST

Kruskal's algorithm is a greedy algorithm that finds a Minimum Spanning Tree (MST) for a connected, undirected weighted graph. It works by sorting all edges by weight, then checking edges one-by-one and adding them to the MST if they do not form a cycle. It uses a Disjoint Set Union (DSU / Union-Find) data structure to keep track of connected components efficiently.

Prim's MST

Prim's algorithm is another greedy algorithm that finds an MST. It builds the tree one vertex at a time, starting from an arbitrary node. At each step, it adds the cheapest possible connection from the already visited tree to an unvisited node, using a priority queue to select the next minimal edge.
