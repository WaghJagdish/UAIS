Linear Search

Linear Search is the simplest search method. It sequentially checks each element of the list until a match is found or the whole list has been searched. It has a complexity of O(n), meaning it scales linearly with the input array size, but requires no pre-sorting.

Binary Search

Binary Search works on a sorted array by repeatedly dividing the search interval in half. It compares the target value to the middle element of the array. If they are unequal, the half in which the target cannot lie is eliminated, and the search continues on the remaining half. This logarithmic complexity O(log n) makes it incredibly efficient for large datasets, though sorting overhead must be factored in.
